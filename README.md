# Homebrew Tap

Custom Homebrew tap for renesenses projects.

## Installation

```bash
brew tap renesenses/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `tune-server` | Multi-room music server with DLNA/UPnP, AirPlay, and streaming services |

### Tune Server

```bash
brew install renesenses/tap/tune-server
brew services start tune-server
```

To update:

```bash
brew update && brew upgrade tune-server
```

## Available Casks

| Cask | Description |
|------|-------------|
| `lanscan` | Fast, native macOS LAN scanner built with SwiftUI |

### LanScan

```bash
brew install --cask lanscan
```
